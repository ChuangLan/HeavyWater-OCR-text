HeavyWater OCR Text Display

This is a WEB APP which can show the corresponding OCR text on a file picture!

Contents of this Project

Your input should be one source file (tiff), and one OCR text file (xml). And once your upload is success, you can see the corresponding text on the picture by clicking areas on the picture. (what else...)

Note: Make sure your image and xml uploaded at the same time!
